PTO V3.0

A simple PTO (Paid time off) calculator that uses how many years you have worked to get correct PTO rate. Then uses how many weeks you have worked so far this year to calculate total PTO to date. Then deducts any TPO that you have taken to date. Results can be returned in hours or days or weeks.

  1. START
  2. DISPLAY "How many have you worked?"
  3. INPUT yearsWorked
  4. DISPLAY "How many weeks have you worked>"
  5. INPUT weeksWorked?
  6. DISPLAY "How many paid time off hours have you used this year?""
  7. INPUT hoursUsed
  8. SET INT totalHours = 40
  9. SET INT totalDay = 8
  10. SET INT checksPerYear = 26
  11. SET (checksPerYear * ptoPerCheck) - hoursUsed
  12. END
