NET Pay Program

A basic net pay calculator. It takes your GROSS pay and then deducts Federal, State, and FICA. It has the option to deduct any additional amount from your check. Then it displays your NET pay.

  1. START
  2. DISPLAY "What is your grossPay"
  3. INPUT grossPay
  4. DISPLAY "Now deducting Federal Taxes."
  5. SET INT fedPay = grossPay * .15
  6. DISPLAY "Now we are deducting ypur State Taxes"
  7. SET INT statePay = grossPay * .05
  8. DISPLAY "Now time to deduct FICA"
  9. SET INT ficaPay = grossPay * .0762
  10. DISPLAY "Put any Additional Deductions"
  11. SET INT additional = 0
  12. INPUT additional
  13. SET INT netPay = grossPay - (fedPay + statePay + ficaPay + additional)
  14. END
